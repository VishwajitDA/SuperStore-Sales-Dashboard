# 📊 SuperStore Sales Dashboard

## Project Overview
This project presents an interactive Power BI dashboard built using the SuperStore dataset. It provides business insights into sales, profit, customer segments, product categories, regions, and monthly sales performance.

## Dashboard Features
- KPI Cards (Sales, Profit, Quantity, Orders)
- Monthly Sales & Profit Trend
- Sales by Category
- Sales by Region
- Sales by Segment
- Sales by State
- Profit by State
- Top Products by Sales
- Interactive Slicers

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
